Key Architecture Features:
  • Event-Driven Architecture: Using Kafka for asynchronous communication between services
  • Caching Strategy: Redis caching to improve performance for frequently accessed data
  • ML Integration: FastAPI service for dynamic pricing using XGBoost model
  • Layered Structure: Clear separation of concerns with specialized components
  • Performance Optimization: Message batching, geospatial indexing, and query optimization
